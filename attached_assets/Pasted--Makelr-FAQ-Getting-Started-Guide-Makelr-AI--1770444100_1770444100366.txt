❓ Makelr FAQ & Getting Started Guide

Makelr는 ‘내가 원하는 일을 대신 해주는 AI 자동화 비서’를 직접 만드는 도구입니다.
코드를 몰라도, 템플릿과 몇 번의 클릭만으로 나만의 AI 워크플로우를 만들 수 있습니다.

🚀 Makelr로 무엇을 할 수 있나요?

Makelr를 사용하면 다음과 같은 일을 자동화할 수 있습니다:

매일 아침 뉴스 / 커뮤니티 / 시장 정보 자동 수집

수집한 자료를 요약, 분석, 리포트로 정리

블로그 글, 보고서, 브리핑 초안 자동 생성

투자 리서치, 트렌드 분석, 콘텐츠 리서치 자동화

내가 원하는 규칙대로 “일하는 AI 비서” 운용

핵심은 이것입니다:

Makelr는 ‘봇을 빌려 쓰는 서비스’가 아니라, ‘내 자동화를 직접 설계하는 도구’입니다.

🧭 시작하기: 6단계로 끝내는 Makelr 사용법
1️⃣ 회원가입 & 로그인

https://makelr.com
 에 접속

회원가입(Sign up) 또는 로그인(Login)

대시보드(Dashboard) 화면이 보이면 준비 완료

2️⃣ LLM API 등록하기 (Claude, OpenAI 등)

Makelr는 당신의 API 키를 사용해 AI를 실행합니다.

좌측 메뉴에서 Settings 클릭

LLM Providers 섹션에서 Add Provider 클릭

다음 항목을 입력합니다:

Name: 예) My Claude API

Provider Type: Anthropic (Claude) / OpenAI 등 선택

API Key: 발급받은 키 붙여넣기

Base URL: 비워두세요 (기본값 사용)

Default Model: 비워두거나 예) claude-3-5-sonnet-latest

Save 클릭

✅ 이제 Makelr가 당신의 AI 모델을 사용할 수 있습니다.

3️⃣ 템플릿으로 자동화 봇 만들기

Dashboard 또는 My Bots 이동

Create Bot / From Template 클릭

원하는 템플릿 선택 (예: Community Research, Investing 등)

봇 이름을 정하고 생성

👉 템플릿은 “이미 잘 설계된 자동화 레시피”입니다.
👉 처음에는 템플릿에서 시작하는 것을 추천합니다.

4️⃣ 소스(Sources) 입력하기

AI가 어디에서 정보를 가져올지 정하는 단계입니다.

좌측 메뉴에서 Sources 클릭

예시:

뉴스 사이트, 블로그 RSS

Reddit, Hacker News, TechCrunch

특정 웹사이트 URL

필요한 소스를 추가 / 수정 / 삭제

저장

👉 여기서 설정한 소스를 기반으로 AI가 자료를 수집합니다.

5️⃣ 콘솔(채팅)로 AI 비서에게 명령하기

좌측 메뉴에서 Console 이동

채팅창에 이렇게 입력해 보세요:

“오늘 수집한 자료 요약해줘”

“이걸 투자 관점에서 정리해줘”

“블로그 글 초안으로 써줘”

AI가 자동으로 작업을 수행합니다

👉 Makelr의 핵심은 **“AI에게 일을 시킨다”**는 감각입니다.

6️⃣ 결과물 검토 & 승인하기

Drafts 또는 Reports 메뉴로 이동

AI가 생성한 초안 확인

수정하거나 그대로 사용

Approve(승인) 또는 Export

👉 여기서 최종 결과물을 사람의 판단으로 컨트롤합니다.
AI는 일하고, 결정은 당신이 합니다.

❓ 자주 묻는 질문 (FAQ)
Q. Makelr는 챗봇 서비스인가요?

아니요. Makelr는 워크플로우 기반 자동화 도구입니다.
채팅은 “명령을 내리는 인터페이스”일 뿐, 핵심은 자동으로 일하는 구조입니다.

Q. 코드를 몰라도 쓸 수 있나요?

네.
템플릿 + 클릭 + 간단한 설정만으로 충분히 사용할 수 있습니다.

Q. API 키는 안전한가요?

Makelr는 당신의 API 키로만 AI를 호출합니다.
키는 외부에 공개되지 않으며, 언제든 교체/삭제할 수 있습니다.

Q. Claude, GPT 둘 다 쓸 수 있나요?

네.
Settings에서 여러 LLM Provider를 등록하고, 상황에 따라 바꿔 쓸 수 있습니다.

Q. 어떤 사람에게 유용한가요?

📊 매일 리서치/자료조사/요약이 필요한 사람

✍️ 콘텐츠 작성, 기획, 정리가 잦은 사람

📈 투자/시장/트렌드 분석을 자동화하고 싶은 사람

🧠 “AI를 도구가 아니라 직원처럼 쓰고 싶은” 사람

🧠 Makelr의 철학

“AI에게 질문하지 말고, 일을 시켜라.”

Makelr는
당신의 시간과 집중력을 대신 벌어주는 자동화 비서를 만드는 도구입니다.
